# Student Portal Backend

## Description
This is the backend application for the Student Portal. It provides RESTful APIs for features like user authentication, announcements, timetables, and more. It also integrates with blockchain technology for secure authentication.

## Features
- Secure user/admin registration and login
- Role-based access control
- Integration with Ethereum blockchain for authentication
- API endpoints for managing announcements, timetables, e-books, and other features
- Real-time chat functionality using WebSocket

## Tech Stack
- **Framework**: NestJS
- **Database**: MongoDB
- **Blockchain**: Ethereum (Smart Contracts in Solidity)

## Installation

### Prerequisites
- Node.js (v14 or higher)
- MongoDB
- Ganache or access to Ethereum testnet
